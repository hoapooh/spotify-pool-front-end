# Spotify Pool Front End 
Listen for free songs on SpotifyPool.
## How to run project
1. First you need to install **pnpm** globally
```bash
npm i -g pnpm
```
2. Then move into Front-End directory
```bash
cd Front-End
```
3. Install dependencies
```bash
pnpm i
```
4. Run project
```bash
pnpm run dev
```
